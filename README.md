# 🧠 Second Brain - Serverless Capture API

A lightweight, serverless API built on AWS to quickly capture ideas, expenses, and tasks directly from your iPhone using Apple Shortcuts. Everything is deployed automatically via GitHub Actions and provisioned using Terraform.

The best part? This project is designed to run entirely within the AWS Free Tier, costing ~$0.00/month.

## 🚀 Features

• **Frictionless Capture**: Voice or text input straight from your iPhone's home screen or Siri.
• **Auto-Categorization**: Basic Python logic to automatically categorize inputs (e.g., "Idea", "Expense", "Task") based on keywords.
• **Infrastructure as Code (IaC)**: 100% reproducible AWS infrastructure using Terraform.
• **CI/CD Ready**: Automated deployments via GitHub Actions.
• **Serverless Architecture**: Scales to zero, no servers to maintain.
• **Secure**: Protected by API Keys to ensure only you can add entries.

## 🏗️ Architecture & Tech Stack

1. **Client**: iOS Shortcuts App (makes an HTTP POST request).
2. **API Layer**: AWS API Gateway (REST API with API Key validation).
3. **Compute**: AWS Lambda (Python 3.12, ARM64 for performance/cost optimization).
4. **Database**: Amazon DynamoDB (Pay-per-request mode).
5. **IaC & CI/CD**: Terraform & GitHub Actions.

## 🛠️ Deployment Instructions

### Option 1: Manual Deployment (Local)

1. Clone this repository:

```bash
git clone https://github.com/yourusername/second-brain-api.git
cd second-brain-api/terraform
```

2. Initialize Terraform:

```bash
terraform init
```

3. Review and apply the infrastructure:

```bash
terraform plan
terraform apply
```

Type `yes` when prompted.

4. Once completed, Terraform will output your `api_url` and `api_key`. Save these for the iOS Setup.

### Option 2: Automated Deployment (GitHub Actions)

1. Go to your GitHub repository settings.
2. Navigate to **Secrets and variables > Actions > New repository secret**.
3. Add the following secrets using your AWS IAM User credentials (ensure the user has permissions for Lambda, API Gateway, DynamoDB, and IAM roles):
	• `AWS_ACCESS_KEY_ID`
	• `AWS_SECRET_ACCESS_KEY`
4. Push your code to the main branch. The GitHub Action defined in `.github/workflows/deploy.yml` will automatically deploy your infrastructure and code to AWS.

*(Note: When using GitHub Actions, you can view your API Key by checking the Terraform Apply step output in the GitHub Actions console).*

## 📱 iOS Shortcut Setup

To connect your iPhone to your new API:

1. Open the Shortcuts app on your iPhone and create a new shortcut (+).
2. Add the **"Ask for Input"** action.
	• Prompt: "What's on your mind?"
	• Type: Text.
3. Add the **"Dictionary"** action.
	• Add a new item: Key = `texto`, Value = `Provided Input` (select the output from step 2).
4. Add the **"Get Contents of URL"** action.
	• URL: [Paste the `api_url` generated by Terraform]
	• Expand the action details:
		• Method: **POST**
		• Headers: 
			• Add `Content-Type` : `application/json`
			• Add `x-api-key` : `[Paste the api_key generated by Terraform]`
		• Request Body: Select **File** and choose the "Dictionary" output from step 3.
5. Name your shortcut something memorable like "Save Idea" or "Second Brain".

**Usage**: Just say "Hey Siri, Save Idea", dictate your thought, and it will be instantly saved to your DynamoDB table!
